# Security Policy

## Reporting a Vulnerability

**Please do not publicly report security vulnerabilities.** Instead, please email security concerns to the maintainers privately.

1. Email your report to the project maintainers
2. Include as much detail as possible about the vulnerability
3. Allow time for a fix before public disclosure

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Security Considerations

When using BLE Scale with Home Assistant:

- Keep Home Assistant updated to the latest stable version
- Ensure your Bluetooth devices are secured
- Use strong passwords for Home Assistant accounts
- Keep your Home Assistant instance on an updated OS
- Consider isolating IoT devices on a separate network

## Dependencies

This integration depends on:
- `bleak` - Pure Python Bluetooth Low Energy library
- Home Assistant core

We regularly update dependencies to address security issues. Please keep the integration updated.
